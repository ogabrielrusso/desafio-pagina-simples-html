O que é HTML?

HTML é uma linguagem de marcação que utiliza tags para estruturar conteúdo web. Criada por Tim Berners-Lee em 1991.

"HTML é a base invisível que sustenta toda a web moderna."

Estrutura Básica

Elementos Fundamentais:

    <!DOCTYPE html>
    <html>
    <head>
        <title>Título</title>
    </head>
    <body>
        Conteúdo
    </body>
    </html>
